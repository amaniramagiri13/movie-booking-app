# 🎬 Movie Booking App

This is a BookMyShow-style movie ticket booking application built using React and Node.js.

## Features
- View available movies
- Select show timings
- Book seats
- Ticket confirmation

## Technologies Used
- React.js
- Node.js
- Express.js
- JavaScript
- HTML
- CSS

## Project Structure

movie-booking-app  
│  
├── client (React frontend)  
└── server (Node.js backend)

## Installation

Clone the repository

git clone https://github.com/amaniramagiri13/movie-booking-app.git

Install dependencies

cd client  
npm install  

cd ../server  
npm install  

Run the project

Frontend:
npm start

Backend:
node server.js
